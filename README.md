## General info
This project is simple list with tasks. You can add tasks, edit them, mark as done and delete them. Every time you refresh, tasks will be deleted.
	
## Technologies
Project is created with:
* Javascript 6
* HTML5
* CSS/SCSS

## Setup
To run this project, download files and open index.html at any code editor with installed saas.
